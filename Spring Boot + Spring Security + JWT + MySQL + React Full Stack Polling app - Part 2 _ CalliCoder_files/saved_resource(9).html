                                                                                                                                                                              Rajeev Singh Mod sameer khan a year ago edited

                                                                                                                                                                              Hi Sameer,

                                                                                                                                                                              1. Json Web Tokens cannot be force expired. Since they are not stored anywhere, you can't mark them as expired.

                                                                                                                                                                              2. Usually, when you're using JWT tokens, you should make it very short-lived (10 minutes) and you can create a long-lived (90 days) refreshToken. When the short-lived accessToken expires, you can get a new accessToken from the server by passing the refreshToken. This concept is used in OAuth2 applications.

                                                                                                                                                                              I've implemented refresh tokens in this Github repository. You can check that out if you're interested. The process is also explained in this StackOverflow answer.

                                                                                                                                                                              That project is part of another tutorial series

                                                                                                                                                                              Cheers,
                                                                                                                                                                              Rajeev
